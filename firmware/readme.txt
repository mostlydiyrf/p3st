ARDUINO LIBRARY NOTES
1.The MultiButton.h and PinButton.h (PinButton is part of MultiButton) libraries have been
modified for use with this project. Use these versions with P3ST_Xiao.ino. To do so, download
the MultiButton folder and place it in your sketchbook/libraries/ folder. Restart the IDE and
open P3ST_Xiao.ino. Confirm that MultiButton appears in the Sketch/Include Library listing.

2. The xiaoRP2040pinNums.h library was written by K7TFC to make Xiao pin references easier.
Download and install the xiaoRP2040pinNums folder as indicated for MultiButton above.
